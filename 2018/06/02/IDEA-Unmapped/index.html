IDEA-Unmapped Spring configuration files found解决办法

springboot 2018/06/02

问题描述

打开IDEA，提示这样一个警告：

解决办法

【File】->【Project Structure…】->【Facets】->【点击】“+” ->【选择】Spring->【点击】module->【点击】OK按钮 ->【点击】“+” ->【选择】配置文件->【点击】OK按钮 -> 【重启】IDEA。
